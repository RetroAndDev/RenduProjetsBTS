À Propos de BrewEngine

Une nouvelle ère du développement homebrew

BrewEngine est un moteur de jeu révolutionnaire créé par Cartouche Studio pour démocratiser le développement sur consoles rétro et modernes. Depuis 2020, notre équipe a investi plus de 50 000 heures dans la création d'une infrastructure solide permettant aux développeurs du monde entier de créer des jeux professionnels sans se battre avec les spécificités techniques de chaque plateforme.

Basé sur une architecture modulaire éprouvée, BrewEngine abstrait les complexités du développement multi-plateforme et fournit des outils de haut niveau tout en conservant un contrôle bas-niveau pour les cas d'usage avancés.

Aujourd'hui, plus de 2000 jeux fonctionnent sur BrewEngine, du port homebrew indie au titre commercial de studio AAA collaborant avec notre équipe.

Simple et Puissant

Créez votre premier jeu en quelques lignes

// main.c - Votre premier jeu BrewEngine #include <brew/engine.h> typedef struct { float x, y; float vx, vy; } Player; void game_init() { Player* player = brew_malloc(sizeof(Player)); player->x = 160; player->y = 120; brew_set_userdata(player); } void game_update(float delta) { Player* player = brew_get_userdata(); if (brew_input_pressed(BREW_KEY_LEFT)) player->vx = -100; else if (brew_input_pressed(BREW_KEY_RIGHT)) player->vx = 100; else player->vx = 0; player->x += player->vx * delta; } void game_draw() { Player* player = brew_get_userdata(); brew_clear(0x000000); brew_draw_sprite("player.png", player->x, player->y); } int main() { brew_init("My First Game", 320, 240); brew_set_callbacks(game_init, game_update, game_draw); return brew_run(); }

Compilez ce code pour n'importe quelle plateforme supportée sans modification !

Architecture & Modularité

Une structure d'engine conçue pour la flexibilité et la performance

🏗️ Core Layer

La couche fondamentale contient les systèmes critiques : gestion mémoire optimisée pour les consoles embarquées, abstraction du rendu graphique compatible avec OpenGL ES 2.0, EGL, et API propriétaires de chaque plateforme.

Tous les allocateurs sont configurable en runtime, permettant des profils mémoire adaptés à chaque console.

⚙️ Game Layer

La couche jeu expose une API haut-niveau : système d'entités avec composants, gestionnaire d'états de jeu, gestionnaire de ressources avec hot-reloading en dev.

Parfait pour les prototypes rapides tout en autorisant l'accès aux couches basses pour les optimisations critiques.

📦 Couche plateforme

Les adaptateurs spécifiques à chaque plateforme : PS1 via PSyQ SDK, PS3 via Cell SDK, Switch via NDK, etc. Chaque port inclut les optimisations natives recommandées.

Automatisé et testé continuellement avec CI/CD à chaque commit.

🔧 Tools & Utils

Suite complète d'outils : convertisseurs d'assets, profilers mémoire et performance, systèmes de scripting Lua optionnels, frameworks de networking.

Tous les outils sont disponibles en CLI et intégrés dans VS Code via notre extension.

Questions Fréquentes

Faut-il payer pour utiliser BrewEngine ?

Non. BrewEngine est 100% gratuit pour les projets personnels et les projets commerciaux générant moins de 50k$/an. Une licence commercial optionnelle existe pour les studios plus grands.

Est-ce vraiment du code source ouvert ?

Oui, 100% sous licence MIT. Vous pouvez modifier, redistribuer, et utiliser commercialement le code. Pas de restrictions d'utilisation ni de viruses GPL.

Puis-je migrer depuis un autre moteur ?

Oui, BrewEngine propose des bindings pour Lua et a une API suffisamment bas-niveau pour être compatible avec les ports existants. Nous avons une équipe pour les migrations complexes (payant).

Quelle est la courbe d'apprentissage ?

Pour les débutants : 1-2 semaines pour comprendre les bases. Pour les développeurs expérimentés : 2-3 jours. Nous fournissons 50+ tutoriels et 100+ examples de code.

Y a-t-il une communauté active ?

Oui ! 5000+ développeurs actifs sur Discord, forums, et GitHub. Nous organisons des game jams mensuelles et des sessions de live-coding. Support communautaire rapide sur les problèmes courants.

Pouvez-vous m'aider pour un projet commercial ?

Absolument. Nous offrons du support prioritaire, du consulting technique, et des builds personnalisés pour les clients enterprise. Contactez enterprise@cartouche-studio.com pour discuter de vos besoins.
